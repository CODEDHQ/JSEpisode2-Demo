### Code Blocks (For Loop)

BLOCK 01 (WHILE LOOP)

```javascript
let index = 0;
let primes = [1, 3, 5, 7, 11, 13, 17, 23];
while (index < primes.length) {
  console.log(primes[index]);
  index++;
}
```

BLOCK 02 (FOR LOOP EQUIVALENT)

```javascript
let primes = [1, 3, 5, 7, 11, 13, 17, 23];
for (let index = 0; index < primes.length; index++) {
  console.log(primes[index]);
}
```
